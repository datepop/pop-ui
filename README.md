Lerna / Vite / Storybook - Monorepo
===================================

This is a monorepo template for a project using Lerna, Vite and Storybook.

## Getting Started

### Install dependencies

```bash
yarn install
```

### Run Storybook

```bash
yarn start:core
```

### Run tests

```bash
yarn test
```

### Run lint

```bash
yarn lint
```
